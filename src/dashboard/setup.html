🔑 Paso 1 de 3 — Cuenta LibreLinkUp

Usa las mismas credenciales que usas en la app LibreLinkUp. El monitor las usará para leer los datos de glucosa de tus familiares.

Elige también una contraseña para el panel de control. Puede ser diferente a tu contraseña de LibreLinkUp y se almacenará derivada con hash (PBKDF2) en config.yaml.

Elige la misma región que tu app LibreLinkUp.

⚠️ Paso 2 de 3 — Umbrales de alerta

Define los valores límite de glucosa (mg/dL) para recibir alertas. Los valores predeterminados son los recomendados para adultos.

🔔 Paso 3 de 3 — Notificaciones (opcional)

Elige cómo quieres recibir alertas de glucosa. Puedes omitir este paso y configurarlo más tarde editando config.yaml.

Sigue estos pasos para configurar las notificaciones de Telegram:

  1. Habla con @BotFather → escribe /newbot → copia el token en el campo de abajo.
  2. Abre tu bot nuevo en Telegram y envíale cualquier mensaje (por ejemplo, Hola).
  3. Haz clic en "📥 Obtener Chat ID" para que se rellene automáticamente.
